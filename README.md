# TIC TAC TOE

---

Play a single player game of tic tac toe. After your move an opposing piece is placed on the grid automatically at random on available spaces. First you choose to play as 'X' or 'O'. Your choice has no effect on the game.

This project uses no libraries or frameworks. It is written with Javascript/jQuery, HTML, and CSS.
 
You can view a live example of this project as a codepen [here](https://codepen.io/DevchamploO/full/Vegmjp/).
